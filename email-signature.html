How to install your email signature

  1. Click inside the white preview box below
  2. Press Ctrl + A to select the signature
  3. Press Ctrl + C to copy
  4. Open Gmail -> Settings -> See all settings -> Signature -> Create new
  5. Click inside the signature box and press Ctrl + V to paste
  6. Save changes
